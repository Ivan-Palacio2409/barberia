# Íconos PWA — FORMA

Reemplaza estos archivos con los íconos reales del negocio:

| Archivo | Uso |
|---------|-----|
| `icon-192.png` | Ícono estándar 192×192 |
| `icon-192-maskable.png` | Ícono maskable 192×192 (con zona segura al 80%) |
| `icon-512.png` | Ícono estándar 512×512 |
| `icon-512-maskable.png` | Ícono maskable 512×512 (con zona segura al 80%) |
| `apple-touch-icon.png` | Ícono iOS 180×180 |
| `favicon.ico` | Favicon clásico 32×32 |

## Diferencia entre "any" y "maskable"

- **any**: el ícono se muestra tal como está, sin recorte.
- **maskable**: Android recorta el ícono en círculo/cuadrado. Deja un 20% de margen
  alrededor del logo para que no quede cortado.

## Herramienta recomendada

https://maskable.app — Previsualiza y exporta íconos maskable desde tu logo.

## Tamaño de fuente sugerido

El logo o letras deben ocupar el 60% del centro del ícono (zona segura) para
garantizar que queden visibles en todos los recortes de Android.
